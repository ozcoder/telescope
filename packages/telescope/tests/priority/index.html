HAR Priority test

This test page is to check the initial priority and final priority of the image requests are recorded correctly in the HAR file. The first sets of images are above-the-fold and so the browser will increase their priority. At the moment, Chromium based browsers increase the priority of the first 5 large (> 10,000 pixels) images to Medium before further increasing them.

The second image is below-the-fold and so should have a Low priority. The third image asks the browser to increase it's priority and so should not have a Low priority. The final image should have the default Low priority that images below-the-fold get.
